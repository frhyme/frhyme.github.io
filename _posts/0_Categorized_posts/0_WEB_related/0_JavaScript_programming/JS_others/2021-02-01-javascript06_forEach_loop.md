## Javascript - forEach loop

- List에서 forEach를 사용하여 각 원소별 동작을 수행하도록 할 수 있습니다.

```html
<!DOCTYPE html>
<html>
    <head>
        <meta charset="utf-8"/>
    </head>
    <body>
        <script>
            var p_lst = ['a', 'b', 'c']
            p_lst.forEach((x) => {console.log(x)})
        </script>
    </body>
</html>
```
