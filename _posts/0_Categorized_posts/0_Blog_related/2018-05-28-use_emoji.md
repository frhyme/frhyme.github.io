## 마크다운에서 emoji 를 쓰고 싶습니다

- 마크다운으로 글을 쓰고 있는데, 가독성을 좋게 하기 위해서는 **이모지**를 좀 더 쓰면 좋을 것 같아요. 쓰려면 뭔가 세팅을 해야 하나? 라고 생각했는데, 그냥 넣으면 다 됩니다. 헤헤

## 쓸만한 이모지 

- 대략 다음만 알아도, 적절하게 쓸 수 있을 것 같아요. 

```plaintext
:+1:
:smile:
:blush:
:flushed:
:arrow_right:
```
