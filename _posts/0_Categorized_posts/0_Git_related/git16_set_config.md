- Git config 내용 정리하기
- 뭘 더 정리해보려고 했는데, 나중에 필요할 때마다 저장하는 것이 더 나을 것 같음.

- `~/.gitconfig` file에 내용을 추가해주면 됩니다.

```sh
git config --list
```
